# VIAPLAST

It's a corporate website that includes a product catalog and an integrated contact form.
Optimized for mobile devices using a responsive layout.
Basic SEO optimization:
  - Custom titles and meta descriptions
  - Google site verification
  - Clean, semantic HTML for better indexing

## WEB-address
Visit VIAPLAST Website: https://viaplast.com.ua/  

## Technologies Used
- HTML5  
- CSS3 / ScSS 
- JavaScript  
- PHP (contact form handling)

## Contact Form
The website includes a contact form. Data is processed with a PHP script located in the `mailer/` directory.  
The file `mailer/smart.php` is excluded from the repository for confidentiality reasons.
